This website talks about front end development

frontend development is the process of building the part of a website or web application that user see and interact with. It involves creating a user interface, button, input, methods and user experience features using web language such as HTML, CSS, JavaScript. The goal of front-end-development is to meet the user requirements and deliver a positive user experience.

Required Language for Frontend Development.

  • HTML
  • CSS
  • JavaScript and it framework
